# TagLang Compiler — Hướng Dẫn Sử Dụng

> **Phiên bản 3.0** | 14 Label | Biên dịch sang C++

---

## TagLang là gì?

TagLang là một project tí hon cho compiler với đặc trưng là **nhãn** (`@label`). Mỗi `@label` mở ra một "ngách" cú pháp riêng, lấy cảm hứng từ nhiều ngôn ngữ lập trình nổi tiếng trên thế giới.

Compiler dịch file `.tl` → file `.cpp` → binary chạy được trên Windows.

---

## Nâng Cấp từ V2 lên V3

V3 khắc phục **toàn bộ 5 giới hạn** của V2 và bổ sung label mới `@func`.

### Fix 1 — `@iter` hỗ trợ gán biến mới

**V2 — bị lỗi:**
```taglang
@iter {
    dem = 0           # ❌ lỗi: 'dem' chưa khai báo
    while dem < 5 {
        dem = dem + 1
    }
}
```

**V3 — hoạt động:**
```taglang
@iter {
    dem = 0           # ✅ khai báo trực tiếp trong @iter
    while dem < 5 {
        dem = dem + 1
        print(dem)
    }
}
```

---

### Fix 2 — `@obj` in nhiều cột trên 1 dòng

**V2 — mỗi field in 1 dòng riêng:**
```
Alice
95
Giỏi
Bob
78
Khá
```

**V3 — dùng `print_row(obj)` in tất cả fields trên 1 dòng:**
```taglang
@obj {
    row1 = { ten: "Alice", diem: "95", loai: "Giỏi" }
    row2 = { ten: "Bob",   diem: "78", loai: "Khá"  }
    print_row(row1)   # ✅ → | Alice | 95 | Giỏi |
    print_row(row2)   # ✅ → | Bob   | 78 | Khá  |
}
```

---

### Fix 3 — `@rscript` vector dùng được trong `@out`

**V2 — không dùng vector trong `@out`:**
```taglang
@rscript { scores <- c(85, 90, 78) }
@out { "Điểm: {scores}" }   # ❌ lỗi: vector không thể in trực tiếp
```

**V3 — tự động tạo biến `_str` dùng trong `@out`:**
```taglang
@rscript { scores <- c(85, 90, 78) }
@out { "Điểm: {scores_str}" }   # ✅ → Điểm: c(85, 90, 78)
```
> V3 tự động tạo `scores_str = "c(85, 90, 78)"` khi khai báo `scores <- c(...)`.  
> Thêm `_str` vào tên vector là xong.

---

### Fix 4 — Label `@func` mới — định nghĩa hàm người dùng

**V2 — không có cú pháp định nghĩa hàm.**

**V3 — thêm label `@func`:**
```taglang
@func {
    func add(x, y) {
        return x + y
    }
    func square(x) {
        return x * x
    }
    func greet() {
        return "Xin chào từ TagLang v3!"
    }

    r1 = add(10, 25)
    r2 = square(7)
    print(r1)          # → 35
    print(r2)          # → 49
    print(greet())     # → Xin chào từ TagLang v3!
}
```
> Hàm được dịch sang **C++ lambda** (`auto add = [&](auto x, auto y) { return x + y; };`).

---

### Fix 5 — Tránh trùng tên biến giữa các label

**V2 — lỗi khi 2 label dùng cùng tên biến:**
```taglang
@data    { x: 100 }
@rscript { x <- 999 }   # ❌ lỗi: 'x' khai báo lại
```

**V3 — tự động thêm hậu tố `_2`, `_3`... khi trùng tên:**
```taglang
@data    { x: 100 }
@rscript { x <- 999 }   # ✅ tự động đổi thành x_2 = 999
@out { "x gốc: {x}" }   # → x gốc: 100
```

## Danh Sách 14 Label (V3)

### 5 Label Gốc (V1)

| Label | Chức năng |
|-------|-----------|
| `@data` | Khai báo dữ liệu: `string`, `int`, `float`, `bool`, `array` |
| `@calc` | Biểu thức toán học: `+ - * / % ^` (lũy thừa) |
| `@logic` | Điều kiện & vòng lặp: `if/else`, `while`, `repeat` |
| `@out` | Xuất dữ liệu theo template `{biến}` |
| `@flow` | Sơ đồ luồng thực thi với mũi tên `->` |

### 8 Label V2 (cảm hứng từ ngôn ngữ khác)

| Label | Ngôn ngữ gốc | Đặc trưng |
|-------|-------------|-----------|
| `@rscript` | **R Language** | `<-` gán giá trị, `c()` vector |
| `@json` | **JavaScript** | `JSON.stringify`, `JSON.parse`, object literal |
| `@ptr` | **Go / C** | `var x *int`, `p = &x`, `*p = 20`, `func f(ptr *int)` |
| `@error` | **Java** | `try/catch/finally`, `throw ExceptionType("msg")` |
| `@array` | **C** | `int arr[5]={1,2,3}`, `char str[]="Hi"`, `char *ptr="Hi"` |
| `@lambda` | **C#** | `(x,y)=>x+y`, `x=>x*x`, `()=>"Hi!"` |
| `@iter` | **Rust** | `for i in 0..10{}`, `for i in 0..=10{}`, `loop{break}` |
| `@obj` | **Ruby / Lua** | `t={}`, `t.name="John"`, `print_row(obj)` *(mới v3)* |

### 1 Label V3 Mới

| Label | Chức năng |
|-------|-----------|
| `@func` | Định nghĩa & gọi hàm người dùng — dịch sang C++ lambda |

## Cài Đặt & Build

### Yêu cầu

- **g++ (MinGW)** đã được thêm vào `PATH`
- Kiểm tra: `g++ --version`

### Build compiler (chỉ làm 1 lần)

```bat
g++ -std=c++17 -O2 src\main.cpp -o taglang.exe
```

> **Lưu ý:** Mỗi khi tải source mới về, phải build lại `taglang.exe`. File `.exe` cũ tự động bị ghi đè.

## Cú Pháp Nhanh — Từng Label

### @data

```taglang
@data {
    ten:    "Nguyễn Văn A"
    tuoi:   20
    gpa:    3.75
    active: true
    scores: [9, 8, 10]
}
```

### @calc

```taglang
@calc {
    tong = a + b * c
    area = pi * r ^ 2     # ^ → pow()
}
```

### @logic

```taglang
@logic {
    if x >= 80 then
        print("Giỏi")
    else
        print("Khá")
    end
    while x > 0 do
        x = x - 1
    end
    repeat 3 times
        print("TagLang!")
    end
}
```

### @out

```taglang
@out {
    "Tên : {ten}   Tuổi: {tuoi}"
    "GPA : {gpa}"
}
```

### @flow

```taglang
@flow {
    start: bat_dau
    bat_dau  -> kiem_tra
    kiem_tra -> ok   if x > 0
    kiem_tra -> loi  if x <= 0
    stop: ok, loi
}
```

### @rscript *(V3: vector_str)*

```taglang
@rscript {
    x    <- 42
    nums <- c(1, 2, 3, 4, 5)
    print(x)
    print(nums)          # in: c(1, 2, 3, 4, 5)
}
@out { "Nums: {nums_str}" }   # ✅ mới v3
```

### @json

```taglang
@json {
    obj person = { name: "Alice", age: 30, score: 95 }
    str s = JSON.stringify(person)
    print(s)
}
```

### @ptr

```taglang
@ptr {
    var x int = 10
    var p *int
    p = &x
    *p = 99
    print(*p)
    func zero(ptr *int) { *ptr = 0 }
    zero(&x)
}
```

### @error

```taglang
@error {
    try {
        throw RuntimeException("Có lỗi!")
    } catch (Exception e) {
        print(e)
    } finally {
        print("Luôn chạy")
    }
}
```

### @array

```taglang
@array {
    int arr[5] = {10, 20, 30, 40, 50}
    char str[] = "Hello TagLang"
    char *ptr  = "Xin chào!"
    print(arr[0])
    print(str)
}
```

### @lambda

```taglang
@lambda {
    add    = (x, y) => x + y
    square = x => x * x
    greet  = () => "Hello!"
    result = add(3, 4)
    print(result)
    print(greet())
}
```

### @iter *(V3: gán biến mới)*

```taglang
@iter {
    dem = 0             # ✅ mới v3: khai báo trực tiếp
    while dem < 5 {
        dem = dem + 1
        print(dem)
    }
    for i in 0..5  { print(i) }
    for i in 0..=5 { print(i) }
    loop { print("một lần") break }
}
```

### @obj *(V3: print_row)*

```taglang
@obj {
    t = {}
    t.name = "John"
    t["tuoi"] = "30"
    person = { name: "Alice", city: "Hanoi" }
    print(t.name)
    print_row(person)   # ✅ mới v3: | Alice | Hanoi |
}
```

### @func *(Mới hoàn toàn V3)*

```taglang
@func {
    func add(x, y) {
        return x + y
    }
    func square(x) {
        return x * x
    }
    func bmi(w, h) {
        return w / (h * h)
    }

    r1 = add(10, 25)
    r2 = square(7)
    r3 = bmi(60, 2)

    print(r1)    # → 35
    print(r2)    # → 49
    print(r3)    # → 15
}
```

## So Sánh V2 vs V3

| Tính năng | V2 | V3 |
|-----------|----|----|
| `@iter` gán biến mới | ❌ phải dùng `@data` | ✅ trực tiếp trong `@iter` |
| `@obj` in nhiều cột | ❌ từng field 1 dòng | ✅ `print_row(obj)` |
| `@rscript` vector trong `@out` | ❌ không dùng được | ✅ dùng `{vec_str}` |
| Định nghĩa hàm | ❌ không có | ✅ label `@func` mới |
| Trùng tên biến | ❌ lỗi runtime | ✅ tự động đổi thành `_2`, `_3`... |
| Số label | 13 | **14** |
| File codegen | Patch nhiều lần | **Rewrite sạch** |

## Mẹo Sử Dụng

- **Xem C++ sinh ra:** `taglang.exe file.tl -o out.cpp` → mở `out.cpp` bằng Notepad hoặc VS Code
- **Debug:** thêm `--tokens` để xem danh sách token
- **Vector trong @out:** dùng `{ten_vec_str}` thay vì `{ten_vec}`
- **Bảng dữ liệu đẹp:** dùng `@out` với `+---+` kết hợp `{biến}` là cách hiệu quả nhất
- **Kết hợp label:** không giới hạn số lượng label trong 1 file
