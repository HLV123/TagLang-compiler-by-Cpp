# TagLang Compiler — Hướng Dẫn Sử Dụng

> 13 Label | Biên dịch sang C++

---

## TagLang là gì?

TagLang là một project siêu tí hon với đặc trưng là **nhãn** (`@label`). Mỗi `@label` mở ra một "ngách" cú pháp riêng, lấy cảm hứng từ nhiều ngôn ngữ lập trình nổi tiếng trên thế giới.

Compiler dịch file `.tl` → file `.cpp` → binary chạy được trên Windows.

---

## Danh Sách 13 Label

### 5 Label Gốc (V1)

| Label | Chức năng |
|-------|-----------|
| `@data` | Khai báo dữ liệu: `string`, `int`, `float`, `bool`, `array` |
| `@calc` | Biểu thức toán học: `+ - * / % ^` (lũy thừa) |
| `@logic` | Điều kiện & vòng lặp: `if/else`, `while`, `repeat` — cú pháp Python |
| `@out` | Xuất dữ liệu theo template `{biến}` |
| `@flow` | Sơ đồ luồng thực thi với mũi tên `->` |

### 8 Label V2 lấy cảm hứng từ ngôn ngữ khác

| Label | Ngôn ngữ gốc | Đặc trưng |
|-------|-------------|-----------|
| `@rscript` | **R Language** | `<-` gán giá trị, `c()` vector |
| `@json` | **JavaScript** | `JSON.stringify`, `JSON.parse`, object literal |
| `@ptr` | **Go / C** | `var x *int`, `p = &x`, `*p = 20`, `func f(ptr *int)` |
| `@error` | **Java** | `try/catch/finally`, `throw ExceptionType("msg")` |
| `@array` | **C** | `int arr[5]={1,2,3}`, `char str[]="Hi"`, `char *ptr="Hi"` |
| `@lambda` | **C#** | `(x,y)=>x+y`, `x=>x*x`, `()=>"Hi!"` |
| `@iter` | **Rust** | `for i in 0..10{}`, `for i in 0..=10{}`, `loop{break}` |
| `@obj` | **Ruby / Lua** | `t={}`, `t.name="John"`, `t["age"]=30`, `respond_to`, `is_a`, `nil` |

## Cài Đặt & Build

### Yêu cầu

- **g++ (MinGW)** đã được thêm vào `PATH`
- Kiểm tra: `g++ --version`

### Build compiler (chỉ làm 1 lần)

```bat
g++ -std=c++17 -O2 src\main.cpp -o taglang.exe
```

### Chạy thử tất cả ví dụ

```bat
.\build.bat
```

> **Lưu ý quan trọng:** Mỗi khi tải source code mới về, phải build lại `taglang.exe` bằng lệnh trên. File `.exe` cũ sẽ được ghi đè tự động — không cần xóa thủ công.

## Sử Dụng Hàng Ngày

### Cách thủ công (3 bước)

```bat
taglang.exe <file.tl> -o out.cpp   ← Dịch .tl sang C++
g++ out.cpp -o program.exe -lm     ← Biên dịch C++
program.exe                         ← Chạy chương trình
```

### Cách nhanh (1 lệnh)

```bat
.\run.bat myfile.tl  : ví dụ như : .\run.bat examples\new_01_rscript_json.tl
```

### Debug — xem danh sách token

```bat
taglang.exe myfile.tl --tokens
```

## Cú Pháp Nhanh — Từng Label

### @data — Khai báo dữ liệu

```taglang
@data {
    ten:    "Nguyễn Văn A"
    tuoi:   20
    gpa:    3.75
    active: true
    scores: [9, 8, 10]
}
```

### @calc — Tính toán

```taglang
@calc {
    tong  = a + b * c
    area  = pi * r ^ 2     # ^ → pow()
    thue  = luong * 10 / 100
}
```

### @logic — Điều kiện & vòng lặp

```taglang
@logic {
    if x >= 80 then
        print("Giỏi")
    else
        print("Khá")
    end

    while x > 0 do
        x = x - 1
    end

    repeat 3 times
        print("TagLang!")
    end
}
```

### @out — Xuất template

```taglang
@out {
    "Tên  : {ten}   Tuổi: {tuoi}"
    "GPA  : {gpa}"
    "Tổng : {tong} điểm"
}
```

### @flow — Sơ đồ luồng

```taglang
@flow {
    start: bat_dau
    bat_dau  -> kiem_tra
    kiem_tra -> ok    if x > 0
    kiem_tra -> loi   if x <= 0
    stop: ok, loi
}
```

### @rscript — R Language

```taglang
@rscript {
    x    <- 42
    ten  <- "TagLang"
    nums <- c(1, 2, 3, 4, 5)
    print(x)
    print(nums)
}
```

### @json — JavaScript JSON

```taglang
@json {
    obj person = { name: "Alice", age: 30, score: 95 }
    str s = JSON.stringify(person)
    print(s)
}
```

### @ptr — Go/C Con trỏ

```taglang
@ptr {
    var x int = 10
    var p *int
    p = &x
    *p = 99
    print(*p)
    func zero(ptr *int) {
        *ptr = 0
    }
    zero(&x)
    print(x)
}
```

### @error — Java Exception

```taglang
@error {
    try {
        throw RuntimeException("Có lỗi xảy ra!")
    } catch (Exception e) {
        print(e)
    } finally {
        print("Luôn chạy đến đây")
    }
}
```

### @array — C Array

```taglang
@array {
    int arr[5]  = {10, 20, 30, 40, 50}
    char str[]  = "Hello TagLang"
    char *ptr   = "Xin chào!"
    print(arr[0])
    print(str)
    print(ptr)
}
```

### @lambda — C# Lambda

```taglang
@lambda {
    add    = (x, y) => x + y
    square = x => x * x
    greet  = () => "Hello!"
    result = add(3, 4)
    print(result)
    print(square(5))
    print(greet())
}
```

### @iter — Rust Iteration

```taglang
@iter {
    for i in 0..5  { print(i) }     # 0 1 2 3 4
    for i in 0..=5 { print(i) }     # 0 1 2 3 4 5
    loop {
        print("chạy một lần")
        break
    }
}
```

### @obj — Ruby/Lua Object

```taglang
@obj {
    t = {}
    t.name    = "John"
    t["tuoi"] = "30"
    person    = { name: "Alice", city: "Hanoi" }
    arr       = {100, 200, 300}
    print(t.name)
    print(arr[1])                   # Lua: 1-indexed
    co_name = person.respond_to(name)
    print(co_name)
}
```

## Thêm Label Mới 

Kiến trúc module giúp thêm label mới rất dễ dàng — chỉ cần 5 bước:

```
Bước 1: src/parsers/ten_moi_parser.h   → Kế thừa BaseParser, implement parse()
Bước 2: src/ast/ast_nodes.h            → Thêm struct node mới
Bước 3: src/label_dispatcher.h         → Thêm 1 dòng: if (label == "ten_moi")
Bước 4: src/codegen/cpp_generator.h   → Thêm hàm genTenMoiBlock()
Bước 5: Rebuild taglang.exe
```

```bat
g++ -std=c++17 -O2 src\main.cpp -o taglang.exe
```

## Giới Hạn của bản V2

| Giới hạn | Ghi chú |
|----------|---------|
| `@iter` không hỗ trợ gán biến mới | Biến dùng trong `while` phải khai báo từ `@data` |
| `@obj` fields chỉ in được từng dòng | Chưa thể in nhiều cột trên cùng 1 hàng |
| `@rscript` vector không dùng được trong `@out` | Chỉ dùng `print()` để in vector |
| Chưa có `@func` / `@class` | Hàm trong `@ptr` là lambda C++ |
| Biến trùng tên giữa các label | Đặt tên khác nhau cho mỗi label |

## HDSD

- **Xem C++ được sinh ra:** `taglang.exe myfile.tl -o out.cpp` rồi mở `out.cpp` bằng Notepad
- **Debug lỗi parse:** thêm `--tokens` để xem danh sách token
- **Kết hợp label:** không giới hạn số lượng label trong 1 file, gọi bao nhiêu cũng được
- **Tránh xung đột biến:** mỗi `@data` và `@calc` đều chia sẻ cùng scope C++ — đặt tên biến khác nhau giữa các block
